# TO-DO-LIST-REST-API with DOMJS

## Summary
The database will be used to assess your understanding of the material provided in the previous few units, in this case `Postgresql`.
This will help you to understand how to set up fullStack without using any framework.


## How do I get started
### Postgresql
- npm init -y
- install express. 
- npm install express 
- npm i dotenv
- npm i pg
- npm i nodemon 
- npm i cors
- npm i body-parser
- npm i cookie-parser

### mongoDB
- npm init -y
- install express. 
- npm install express 
- npm i dotenv
- npm i mongoose
- npm i nodemon 
- npm i cors
- npm i body-parser
- npm i cookie-parser


## Tech Stacks
Postgresql, Vanilla JS with DOM
